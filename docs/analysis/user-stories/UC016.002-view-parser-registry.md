# User Story UC016.002: View Parser Registry

| Attribute | Value |
|-----------|-------|
| **Story ID** | UC016.002 |
| **Epic** | Financial Management |
| **Related UC** | UC016 |
| **Priority** | SHOULD HAVE |
| **Story Points** | 2 |

**As an** ADMIN **I want to** view the list of available import parsers **so that** I know which bank formats are supported and their current status.

## Acceptance Criteria

**AC1:** Settings page (Transactions → Settings) includes a "Import Parsers" sub-section listing all parsers (active and inactive) in a table: code, label, format badge (CSV / PDF), bank hint, active status, description.

**AC2:** Inactive parsers shown with a greyed-out "Inactive" badge. Not available in the import parser dropdown (UC016.001).

**AC3:** No create, edit, or delete actions in the UI — parsers are managed by developers via Flyway seed data and deployed with the application.

**AC4:** Table accessible to ADMIN only.

**Endpoint:** `GET /api/v1/import-parsers?activeOnly=false` — returns all parsers including inactive.
