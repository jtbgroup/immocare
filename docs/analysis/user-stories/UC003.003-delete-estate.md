# UC003.003 — Delete Estate

## Overview

| Attribute | Value |
|---|---|
| **ID** | UC003.003 |
| **UC** | [UC004_ESTATE_PLACEHOLDER — Manage Estates](../use-cases/UC004_manage_estates.md) |
| **Actor** | PLATFORM_ADMIN |
| **Priority** | MUST HAVE |

## User Story

**As a** PLATFORM_ADMIN **I want to** delete an estate **so that** I can remove obsolete or test estates.

## Acceptance Criteria

- AC1: Blocked if estate contains any buildings → HTTP 409 with `buildingCount`.
- AC2: On success: estate and all cascade-eligible data permanently deleted, HTTP 204.
- AC3: Confirmation dialog in the UI showing estate name in bold and "This action cannot be undone."
- AC4: Cancel: dialog closes, estate not deleted.

## Endpoint

`DELETE /api/v1/admin/estates/{id}`

## Response

HTTP 204 — No content

## Business Rules

- BR-UC004_ESTATE_PLACEHOLDER-09: Cannot delete an estate that contains buildings
