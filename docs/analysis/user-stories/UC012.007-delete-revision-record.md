# User Story UC012.007: Delete Revision Record

| Attribute | Value |
|-----------|-------|
| **Story ID** | UC012.007 |
| **Epic** | Fire Safety Management |
| **Related UC** | UC012 |
| **Priority** | SHOULD HAVE |
| **Story Points** | 1 |

**As an** ADMIN **I want to** delete a revision record **so that** I can correct data entry mistakes.

## Acceptance Criteria

**AC1:** Each revision row in the history table shows a delete icon (🗑).
**AC2:** Click icon → inline confirmation: "Delete this revision record? This action cannot be undone."
**AC3:** Confirm → revision deleted; row disappears; revision count badge on the card decrements; latest revision date reverts to the next most recent date (or "Never inspected" if no revisions remain).
**AC4:** Cancel → no deletion, row remains.

**Endpoint:** `DELETE /api/v1/fire-extinguishers/{extId}/revisions/{revId}` — HTTP 204.
