**As an** ADMIN **I want to** see if PEB scores have improved over time **so that** I can evaluate renovation impact.

## Acceptance Criteria

**AC1:** History D→C→B → improvement indicators: D→C: ↑ "Improved" (green), C→B: ↑ "Improved" (green).
**AC2:** Score B→C (degradation) → ↓ "Degraded" (red).
**AC3:** Two consecutive B scores → − "No change" (grey dash).
**AC4:** First score F (2018), current B (2024) → summary "Improved by 4 grades over 6 years".
