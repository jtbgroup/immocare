# User Story US024: Track Rent Increases Over Time

| Attribute | Value |
|-----------|-------|
| **Story ID** | US024 |
| **Epic** | Rent Management |
| **Related UC** | UC005 |
| **Priority** | COULD HAVE |
| **Story Points** | 3 |

**As an** ADMIN **I want to** track rent increases over time **so that** I can analyze rental income trends.

## Acceptance Criteria

**AC1:** History €750 (2022) → €800 (2023) → €850 (2024) → trend indicators shown: 2022→2023: +€50 (+6.67%) ↑, 2023→2024: +€50 (+6.25%) ↑.
**AC2:** First rent €750, current €850 → summary "Total increase: +€100 (+13.33%) over 2 years".
**AC3:** Rent decreased €900→€850 → shows "-€50 (-5.56%)" ↓.
**AC4:** (Optional) Line chart shows rent evolution over time.
