# UC003.008 — Edit Member Role

## Overview

| Attribute | Value |
|---|---|
| **ID** | UC003.008 |
| **UC** | [UC004_ESTATE_PLACEHOLDER — Manage Estates](../use-cases/UC004_manage_estates.md) |
| **Actor** | MANAGER, PLATFORM_ADMIN |
| **Priority** | MUST HAVE |

## User Story

**As a** MANAGER **I want to** change a member's role **so that** I can promote a viewer or demote a manager.

## Acceptance Criteria

- AC1: Editable field: `role` (MANAGER / VIEWER).
- AC2: Cannot change own role → HTTP 409 "You cannot change your own role."
- AC3: Cannot demote the last MANAGER of an estate → HTTP 409 "Cannot remove the last manager of an estate."
- AC4: Only MANAGER or PLATFORM_ADMIN can edit roles.
- AC5: Change takes effect immediately.

## Endpoint

`PATCH /api/v1/estates/{id}/members/{userId}`

## Request Body

```json
{
  "role": "MANAGER"
}
```

## Response

HTTP 200 — `EstateMemberDTO`

## Business Rules

- BR-UC004_ESTATE_PLACEHOLDER-02: At least one MANAGER must remain after the role change
- BR-UC004_ESTATE_PLACEHOLDER-04: User cannot change their own role
